Команды в подсистеме Ubuntu

Команда Как работает?
ls Показывает файлы в открытой папке
mkdir Создает папку
cd Открывает папку
touch Создает файл txt
cd ../ Закрывает папку
rmdir Удаляет папку
rm Удаляет файл
rm -r (названия файлов) Удаляет несколько файлов
code (названия файлов) Открывает несколько файлов в программе
